# Invoice Dashboard — Hiring Lab

## Requirements

- Node.js 18+
- npm 9+

## Setup

```bash
npm install
npm run dev
```

## Run tests

```bash
npm run test
```

## Type check

```bash
npx tsc --noEmit
```

## CI

Push to any branch to trigger the CI pipeline.
The pipeline runs type checks, tests, and validates commit discipline.
